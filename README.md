# Athletic Facility & Event Resource Optimizer

## Project Overview
This project is an analytical management tool designed to optimize operations and financial tracking for large-scale athletic facilities and competitive sporting events. It mirrors enterprise resource management by combining inventory lifecycle forecasting with dynamic budget optimization.

### Key Features
* **Predictive Maintenance (SAS):** A statistical forecasting model that analyzes facility foot traffic, equipment usage rates, and training-focus metrics to predict wear-and-tear. It automatically flags high-risk inventory for immediate servicing, preventing equipment failure.
* **Budget Optimization Dashboard (VBA & MS Excel):** An automated financial tracker designed for managing large-scale event budgets (e.g., inter-collegiate trials). It instantly calculates cost variances, formats financial data, and actively flags deficit categories (like participant accessories or operational costs) to prevent overspending.

## Tools Used
* **SAS:** Statistical Modeling & Lifecycle Forecasting
* **VBA / MS Excel:** Financial Dashboard Automation, Variance Analysis, & Data Visualization
